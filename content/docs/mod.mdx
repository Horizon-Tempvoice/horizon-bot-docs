As a channel owner, you can promote members to **channel moderators**. Mods can help you manage your channel and use a configurable set of commands on your behalf.

Use `/mod add @user` to add a moderator and `/mod remove @user` to remove one.

<Callout title="Note" type="info">
    Only the channel owner can add or remove moderators. A channel can have a maximum of **7** moderators.
</Callout>

## Customizing Permissions Per Channel

Channel owners can use `/mod permissions` to customize which commands their mods are allowed to use. A selection menu will appear with all available permissions. Select the ones you want to grant.

<Callout title="Note" type="info">
    Depending on the server admin's **Enforce Setting**, channel owners can either only restrict the delegated permissions or freely choose from all available commands.
</Callout>

## Server Admin Configuration

<Callout title="Where to find this?" type="info">
    Open the [Dashboard](https://dashboard.horizon-bot.me), select your server, pick the correct Creator Channel and navigate to the **Access Tab**.
</Callout>

![Channel Mod Configuration](/img/content/channel_mod_config.png)

Server admins can configure moderator behavior per generator in the **Channel Moderators** section of the Access tab.

| Setting | Description |
|---|---|
| **Moderator Permissions** | Select which commands are available to channel moderators. Changes apply to all channels created by this generator. |
| **Enforce Setting** | When enabled, channel owners can only pick from the permissions selected above. When disabled, the selection acts as a default and owners can freely adjust via `/mod permissions`. |
| **Protect Moderators** | When enabled, channel moderators cannot use mod-level commands (kick, block) against each other. The channel owner is always exempt from this restriction. |
