## What is a temporary voice channel and how do I benefit from it?

Think of it as a voice channel that only exists when someone actually needs it. When a user joins a <strong style={{ color: BRAND_COLOR }}>Join to Create-Kanal</strong>, Horizon spins up a private room for them - and once everyone leaves, it's gone. No leftovers, no cleanup needed.

* **No more empty channels:** The channel list only shows channels that are actually being used.
* **Members manage their own space:** As the channel owner, you can rename it, lock it, set a user limit and more, either via `/commands` or the <strong style={{ color: BRAND_COLOR }}>Horizon-Panel</strong>. No admin required.
* **Fully configurable for admins:** Server admins decide exactly what members are allowed to change in their channels.
* **Scales automatically:** Whether it's 2 people or 200, the server just keeps itself clean.
* **Private by default:** Each room belongs to the person who created it, for as long as they need it.

No clutter. No manual work. It just handles itself.
