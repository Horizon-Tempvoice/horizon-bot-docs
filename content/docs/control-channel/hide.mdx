---
title: Hide Channel
description: Hide your channel from everyone except trusted members.
icon: EyeOff
---

Use `/hide` to make your channel invisible. Only users you have granted access with `/trust` can still see and join it.

<Callout title="Channel Moderators" type="info">
    Channel moderators can use this command if explicitly enabled by the server admin in the dashboard.
</Callout>
