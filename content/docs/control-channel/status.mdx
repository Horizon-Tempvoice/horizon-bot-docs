---
title: Set Status
description: Set a short status message on your temporary channel.
icon: MessageSquare
---


Use `/status set` to add a status text to your channel that is visible to all members in the channel list. Use `/status remove` to clear it.

<Callout title="Channel Moderators" type="info">
    Channel moderators can use this command if explicitly enabled by the server admin in the dashboard.
</Callout>
