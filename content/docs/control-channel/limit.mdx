---
title: Channel User Limit
description: Limit the maximum amount of users can join to your channel
icon: Users
---

![Channel Limit](/img/content/channel_limit_command_en.png)

In this example, the channel will change its user limit from ``unlimited`` to ``4``.

<Callout title="Note" type="idea">
    This overwrites the default Temporary Channel User Limit configured in the [Dashboard](https://dashboard.horizon-bot.me) if the user saves the channel to a
    ``Session profile``!
</Callout>
