---
title: Block User
description: Block a user from joining your temporary channel.
icon: UserX
---


Use `/block @user` to permanently prevent a user from joining your channel. If they are currently inside, they will be disconnected. Blocked users also cannot be seen in the channel list if it is hidden.

<Callout title="Note" type="info">
    You cannot block yourself or the channel owner. Some members may also be protected by server admins and cannot be blocked (e.g. server moderators).
</Callout>

<Callout title="Channel Moderators" type="info">
    Channel moderators can also use this command. Server admins can disable this in the dashboard. By default, moderators cannot block each other (see [Protect Moderators](/mod#server-admin-configuration)).
</Callout>
