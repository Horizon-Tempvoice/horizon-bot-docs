---
title: Channel Region
description: Set the voice server region for your temporary channel.
icon: Globe
---

Use `/region` to select the voice server region for your channel. Choose from a list of available regions or select ``Automatic`` to let Discord decide.

Available regions: Automatic, Brazil, Hong Kong, India, Japan, Milan, Rotterdam, Russia, Singapore, South Africa, South Korea, Sydney, US Central, US East, US South, US West.

<Callout title="Channel Moderators" type="info">
    Channel moderators can also use this command. Server admins can disable this in the dashboard.
</Callout>
