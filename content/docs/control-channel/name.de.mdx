![Kanal umbenennen](/img/content/channel_rename_command_en.png)

In diesem Beispiel wird der Kanalname von ``Fabi's Channel`` auf ``Your awesome name`` geändert.

<Callout title="Hinweis" type="idea">
    Das überschreibt den Standard-Kanalnamen aus ``/generator config``, sobald du den Kanal in einem
    ``Session-Profil`` speicherst.
</Callout>
