In this example, the channel will change its name from ``Fabi's Channel`` to ``Your awesome name``

<Callout title="Note" type="idea">
    This overwrites the default Temporary Channel Name that is configured in ``/generator config`` if the User save the channel to a
    ``Session profile``!
</Callout>
