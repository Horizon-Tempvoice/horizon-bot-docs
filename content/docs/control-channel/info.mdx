![Channel Info](/img/content/info_command_en.png)

Use `/info` to get an overview of your current temporary channel.

| Field | Visible to |
|---|---|
| Owner | Everyone |
| Created | Everyone |
| Privacy (locked / hidden) | Everyone |
| Moderators | Everyone |
| Trusted users | Owner & Mods |
| Blocked users | Owner & Mods |
