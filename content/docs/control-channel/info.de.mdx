---
title: Kanal-Info
description: Zeige Informationen über deinen temporären Sprachkanal an.
icon: Info
---

![Kanal-Info](/img/content/info_command_en.png)

Mit `/info` bekommst du eine Übersicht über deinen aktuellen temporären Kanal.

| Feld | Sichtbar für |
|---|---|
| Besitzer | Alle |
| Erstellt | Alle |
| Privatsphäre (gesperrt / versteckt) | Alle |
| Moderatoren | Alle |
| Vertraute Nutzer | Besitzer & Mods |
| Blockierte Nutzer | Besitzer & Mods |
