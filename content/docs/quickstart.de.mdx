# 2. Horizon einrichten

Mit dem `/setup`-Befehl richtest du Horizon auf deinem Server ein. Klick auf `Setup Horizon` und der Bot erstellt dir automatisch einen <strong style={{ color: BRAND_COLOR }}>Join to Create</strong>-Kanal sowie ein <strong style={{ color: BRAND_COLOR }}>Horizon-Panel</strong>.

# Generator-Kanal

Diese speziellen Sprachkanäle erstellen automatisch einen temporären Kanal, sobald jemand beitritt. Du kannst mehrere Generator-Kanäle einrichten, indem du die Einrichtung einfach wiederholst. Alle davon lassen sich unabhängig anpassen.

<Callout title="Hinweis" type="warn">
  Du kannst maximal 3 Generator-Kanäle pro Server haben. Wenn du einen Generator
  löschst, wird auch der zugehörige Eintrag in der Bot-Datenbank entfernt.
</Callout>

# Tempvoice-Panel

Das Tempvoice-Panel, <em>auch <strong style={{ color: BRAND_COLOR }}>Horizon-Panel</strong> genannt</em>, ist eine Discord-native Oberfläche, über die du deine temporären Sprachkanäle verwalten kannst als eine Alternative zu den Slash-Commands.
