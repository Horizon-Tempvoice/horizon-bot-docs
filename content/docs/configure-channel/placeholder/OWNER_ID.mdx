---
title: "{OWNER_ID}"
description: Inserts the Discord user ID of the channel owner.
---

`{OWNER_ID}` is replaced by the Discord user ID of the channel owner. This is a unique numeric identifier assigned to every Discord account.

A template like `` 🎙・{OWNER_ID} `` would produce channels such as:

- ``🎙・123456789012345678``
