---
title: "{OWNER_NICK}"
description: Inserts the server display name of the channel owner.
---

`{OWNER_NICK}` is replaced by the channel owner's display name on the server. If they have a server nickname set, that is used. Otherwise it falls back to their account username.

A template like `` 🎙・{OWNER_NICK} `` would produce channels such as:

- ``🎙・Fabi``
- ``🎙・ProGamer``
- ``🎙・john_doe``

This is also the default placeholder used in the channel name template.
