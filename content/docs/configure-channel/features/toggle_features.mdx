---
title: Toggle Features
description: Control which features are available to channel owners in their temporary channel.
---

As a server admin you can decide which features channel owners are allowed to use. Disabled features are hidden from channel owners and their corresponding commands will not work.

<Callout title="Where to find this?" type="info">
    Open the [Dashboard](https://dashboard.horizon-bot.me), select your server, pick the correct Creator Channel and navigate to the **Feature Tab**.
</Callout>

![Toggle Features](/img/feature-tab-docs/toggle_features.png)

<Callout title="Note" type="idea">
    If a feature is disabled, the user's saved configuration for that specific feature will not be applied.
</Callout>
