---
title: Save Session
description: Save your current channel configuration to a slot.
icon: Save
---

Use `/session save [slot] [label]` to save your current channel state. This includes name, user limit, lock/hide state, trusted users, blocked users, and moderators.

The optional `label` helps you identify the slot (max. 32 characters).

<Callout title="Note" type="info">
    You can save up to **3 slots**. Saving to an existing slot overwrites it.
</Callout>
