Session profiles let you save your current channel configuration, including name, user limit, lock/hide state, trusted users, blocked users, and moderators, and automatically apply it the next time you create a channel.

You can have up to **3 saved slots**, each with an optional label.

<Callout title="Note" type="info">
    Server admins can disable the session feature. If the commands below don't work, sessions may not be available on your server.
</Callout>
